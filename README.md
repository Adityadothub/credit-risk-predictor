# credit-risk-predictor
Machine Learning web app for credit risk prediction using Streamlit

This project was inspired by my experience working on a credit card delinquency analysis job simulation.

During that work, I explored how financial institutions assess the likelihood of customers defaulting on payments. 
The simulation provided insights into key risk indicators such as credit amount, repayment duration, and financial behavior.

While this application is not a direct implementation of that project, certain aspects—such as feature selection 
and the overall problem framing—are influenced by that experience.

This project extends those ideas into a machine learning-based system that provides an interactive interface 
for credit risk prediction.
